# Matrix Pokémon - Game Manual Storyboard

This document provides the structured "first step" for the game manual, as requested in `job_list.md`. It categorizes the source photographs and provides a "Manga Translation" guide for each stage of the instructions.

## Art Concept (Recap)
- **Style**: Manga/Comic with clear black outlines (Yang Style).
- **Colors**: Pale watercolor palette focusing on content.
- **Themes**: Pokémon + Matrix Universum.
- **Audience**: 6-year-olds kids (IKEA-style simplicity).

---

## Stage 1: Game Setup
- build deck with 60 card
- 6 price cards
- 54 matrix cards

## Stage 2: Starting the Game
- setup 6 x 9 Matrix
- 6 price cards
- discard pile

## Stage 3: Gameplay & Interaction
- turn card face up
- move active pokemon
- move other pokemon, play trainer
- fight
- clean up

## Stage 4: Fight and Victory & End of Game
- -10 damage per row --> front row = -0 --> 2nd row= - 10 --> 3rd row = -20 --> 6th row = -50

- + defences per row   --> front row = +0 --> 2nd row= - 10 damage, 3rd row = -20  and 6th row = -50

- Attack in 6th row, defender in 6th row = -100 damage
- Attach in 1st row and defender in 1st row = - 0 damage 0 full damage
- if HP of pokemon = 0 remove all from game to discard, get winpoitns
